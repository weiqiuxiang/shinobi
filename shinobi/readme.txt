本作品はHAL東京在学中で、3年生の時制作した作品です。
キャッチコピー : 効率的に3Dゲームを作る

---操作説明---
コントローラ、キーボードやマウスを使用して操作する

<<タイトル画面>>
キーボードやマウスを使用して操作する
項目の選択 : マウスホイール / ↑キー / ↓キー
項目の確定 : マウス左ボタン / ENTERキー

<<ゲーム画面>>
XBOX360のコントローラを使用してプレーします。
キーボードもサポートしています。
移動     : W/A/S/D
走る     : SPACEキー + 移動
回避     : SPACEキー
攻撃     : Jキー
武器変形 : Kキー
敵にロックオン : ENTERキー

<<マップエディター画面>>
マウスを使用して操作する
カメラスケール : マウスホイール
カメラ回転     : マウス右ボタンを押しながらマウスを移動する
カメラ移動     : マウスホイールを押しながらマウスを移動する
オブジェクトの配置とGUIの操作 : マウス左ボタン

<<モーションエディター画面>>
マウスを使用して操作する
カメラスケール : マウスホイール
カメラ回転     : マウス右ボタンを押しながらマウスを移動する
カメラ移動     : マウスホイールを押しながらマウスを移動する
モデルの操作とGUIの操作 : マウス左ボタン
